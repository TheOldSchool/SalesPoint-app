# SalesPoint

## Project setup
### Install Vue Cli 3
```
npm install -g @vue/cli
```

### Install Vue plugin on VSCode
```
Vetur
```
### Install dependencies
```
With Yarn
yarn install

With npm
npm install
```

### Compiles and run the server app
```
npm run serve
```

### See web page on browser with
```
http://localhost:8080
```
